## 💻 Projeto

Aplicação simples com o propósito de estabelecer a criação de um deploy workflow de aplicação Blazorr usando o host do github pages.

## 🏗️ Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- .NET 7.0
- BlazorWebAssembly
